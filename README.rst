unit-bot
========

Unit conversion bot for reddit.

Usage
=====

Create ``creds.py`` file in ``unit-conversion-bot/`` directory.

Example of ``creds.py``:

.. code-block:: python

    USERNAME = 'my-bot'
    PASSWORD = 'hunter2'

    USER_AGENT = 'unit bot'

Run it:

.. code-block:: bash

    make
    make run
